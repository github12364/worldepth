Naming schema: yourname/branchname

To see what branch you're on:
git status

To make and checkout a new branch:
git checkout -b NameOfBranch

To push new changes:
1. Save your changes.
2. git add .
3. git commit -m "description of the changes"
4. git push
5. If your upstream is not set up yet, git will give you a suggested command in order to push

To pull from the remote (online) branch:
git pull

To get changes made to the remote master branch:
git pull origin master
